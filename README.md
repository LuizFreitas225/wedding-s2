---

## 🖥️ Como abrir localmente

Não é obrigatório instalar nada. Basta abrir o arquivo `index.html` no navegador ou servir a pasta com um servidor HTTP simples:

```bash
# Na pasta do projeto
python3 -m http.server 8080
```

Depois acesse `http://localhost:8080`.

## 🌐 Publicação

Se você hospedar no GitHub Pages (ou outro serviço estático), lembre de atualizar no `index.html` as meta tags `og:url` e `og:image` com a URL real do site — assim o compartilhamento nas redes fica com a capa certinha.
